## The and Operators Truth Table
- True and True = True
- True and False = False
- False and True = False
- False and False - False

## The or Operators Truth Table
- True or True = True
- True or False = True
- False or True - True
- False or False - False

## The not Operators Truth Table
- not True = Flase
- not False = True
